# Forest Fire Detection System

## 🔥 Overview
This project detects forest fire using sensors and wireless communication.

## ⚙ Components Used
- Arduino Nano (Sender)
- ESP32-C3 (Receiver)
- DHT11 Temperature Sensor
- MQ-2 Smoke Sensor
- Flame Sensor / Button (for simulation)
- HC-12 Wireless Module

## 🚀 Working
- Sensors detect temperature, smoke, and fire
- Arduino sends data via HC-12
- ESP32 receives data and triggers LED alert

## 💡 Features
- Real-time fire detection
- Wireless communication
- LED alert system

## 🧪 Simulation
In Wokwi, serial communication is used instead of HC-12.
